I. 미분 
1. 변수의 움직임에 따른 함수 값의 변화를 측정하기 위한 도구, 최적화에서 제일 많이 사용하는 기법
2. 최근엔 미분을 손으로 직접 계산하는 대신 컴퓨터가 계산해줄 수 있음

- 접선의 기울기
  = 1. 미분은 함수 f 의 주어진 점 (x, f(x)) 에서 접선의 기울기를 구함
  = 2. 한 점에서 접선의 기울기를 알면 어느 방향으로 점을 움직여야 함수값 증가/감소 여부를 알 수 있음
    -> 1. 증가시키고 싶으면 미분값 더하고, 감소시키고 싶으면 미분값 감소
    -> 2. 미분값이 음수라서 x + f(x) < x 는 왼쪽으로 이동하여 함수 값이 증가
    -> 3. 미분값이 양수라서 x - f(x) > x 는 오른쪽으로 이동하여 함수 값이 감소

  = 3. 미분값을 더하면 경사상승법 (Gradient Ascent), 함수의 극대값의 위치를 구할 때 사용
  = 4. 경사상승/하강 방법은 극값에 도달하면 움직임을 멈춤

- 변수가 벡터일 경우
  = 1. 미분 (Differentiation) 은 변수의 움직임에 따른 함수 값의 변화를 측정하기 위한 도구
    -> 최적화에서 제일 많이 사용하는 기법

  = 2. 벡터가 입력인 다변수 함수의 경우 편미분 (Partial Differentiation) 을 사용
  = 3. 각 변수 별로 편미분을 계산한 Gradient Vector 를 이용하여 경사하강 / 상승법에 사용 가능
  = 4. ∇ -> Nabla 라고 부름
