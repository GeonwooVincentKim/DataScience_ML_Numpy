# Data Mining
  ## 정의
    ### 대량 Dataset 의 처리 및 탐색을 위한 분석에 사용되는 Computer 지원 기법
    ### Data Mining 도구와 방법을 사용해 조직은 Data 에 숨겨진 Pattern 과 관계를 찾을 수 있으며, 원시 Data 를 실용적인 지식으로 변환됩니다.
  
  ## Data Mining Process
    -> 1. Business 목표 설정
    -> 2. Data 준비
    -> 3. Model 구축 및 Pattern Mining
    -> 4. 결과 평가 및 지식 활용
