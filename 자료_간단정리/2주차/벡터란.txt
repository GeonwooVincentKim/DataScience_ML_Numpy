벡터란?
- 공간에서 한 점을 나타냄
- 원점으로부터 상대적 위치를 표현함
- 숫자를 곱해주면 길이만 변함
  = 1보다 크면 길이나 늘어남
  = 1보다 작으면 길이가 줄어듬
  = 0보다 작으면 반대 방향이 됨

- 숫자를 원소로 가지는 List or Array
- 벡터끼리 같은 모양을 가지면 덧셈, 뺄셈, 성분곱 (Hadamard Product) 계산 가능

벡터의 덧셈
- 두 벡터의 덧셈은 다른 벡터로부터 상대적 위치이동을 표현함

벡터의 뺄셈
- 원래 X라는 벡터가 움직이는 반대 방향으로 움직이는 벡터

벡터의 노름
- 벡터의 노름 (norm) 은 원점에서부터의 거리를 말함
- L1 Norm, L2 Norm -> 임의의 차원 d, 거리 계산, 구성 성분 원소 개수에 상관없이 계산 가능
- L1 Norm 은 각 성분의 변화량의 절대값을 모두 더함
- L2 Norm 은 피타고라스 정리를 이용해 유클리드 거리를 계산함
- 각 구성 성분의 절대 값을 먼저 계산 -> 더함 (Numpy 활용하여 Sum 값 계산 - 절대값 기준)
- L2 Norm 은 np.linalg.norm 을 이용해도 구현 가능

Norm??
- Norm 의 종류에 따라 기하학적 성질이 달라짐

두 벡터 사이의 거리
- L1, L2 Norm 을 이용해 두 벡터 사이의 거리를 계산 가능
- 두 벡터 사이의 거리를 계산할 때는 벡터의 뺄셈을 이용
- 뺄셈을 거꾸로 해도 거리는 같음

두 벡터 사이의 각도
- L2 Norm 에서만 가능
- 제2 코사인 법칙에 의해 두 벡터 사이의 각도를 계산 가능
- 분자를 쉽게 계산하는 방법이 내적 (Internal -> Inner)

내적 해석 방법
- 내적은 정사영 (orthogonal projection) 된 Vector 의 길이와 관련 있음
- 정사영 -> X Vector 와 Y Vector 가 그림 상에 있으며, 빛을 쏘았을 때 X Vector 의 Y Vector 상에서의 그림자
- Proj(x) -> Y Vector 로 정사영된 X Vector 의 그림자
- Proj(x) 의 길이는 코사인법칙에 의해 ||x|| cos제타가 됨
- 내적은 정사영의 길이를 Y Vector 의 길이 ||y|| 만큼 조정한 값 -> 유사도 (Similarity) 측정 시 사용 가능
